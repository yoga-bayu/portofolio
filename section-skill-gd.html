Graphic Design

Selain saya belajar tentang IT, saya juga belajar graphic design atau desain grafis. Mungkin dengan latar belakang seni dan desain, saya memilki keinginan untuk menciptakan karya visual yang menarik dan memikat.


Pengalaman saya dalam menggunakan berbagai perangkat lunak desain seperti canva, corel draw, dan lain-lain memungkinkan saya untuk menghasilkan desain yang kreatif dan profesional.


Dengan keinginan untuk berkreasi dan inovasi, saya siap untuk mengembangkan skill design saya, juga saya akan tetap terus belajar terkait design. Saya berharap, supaya saya dapat mengimplentasikan atau menerapkan skill design yang saya miliki, supaya berguna untuk diri saya pribadi ataupun orang lain.

Foto-foto di atas merupakan beberapa project design yang pernah saya buat, seperti poster, iklan postingan, promosi dan lain-lain. Saya juga pernah berkesempatan menjadi bagian dari tim sebuah perusahaan yang memproduksi kasur yakni PT. Aneka Busa Indonesia, dimana saya pernah berkontribusi membuat design untuk iklan promosi penjualan mereka.
